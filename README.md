# animacion3d
# Animación 3D con Java y JOGL

Este repositorio contiene un proyecto de graficación por computadora desarrollado en **Java** utilizando la librería **JOGL (Java Binding for the OpenGL API)**.

El objetivo principal es demostrar el renderizado de objetos y transformaciones en un entorno tridimensional.

## Características
* Renderizado de objetos 3D primitivos.
* Implementación de transformaciones geométricas (rotación, traslación, escalado).
* Uso del ciclo de animación estándar de OpenGL.

## Tecnologías Utilizadas
* **Lenguaje:** Java
* **Librería Gráfica:** JOGL (OpenGL)
* **IDE Recomendado:** Visual Studio Code
* **Control de Versiones:** Git / GitHub

## Requisitos Previos
Para ejecutar este proyecto necesitas tener instalado:
* [Java JDK](https://www.oracle.com/java/technologies/downloads/) (Versión 8 o superior).
* Librerías de JOGL configuradas en tu Classpath.
* Visual Studio Code con el "Extension Pack for Java".

## Instalación y Uso

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/LuisHdz089/animacion3d.git](https://github.com/LuisHdz089/animacion3d.git)
    ```

2.  **Abrir en VS Code:**
    Abre la carpeta del proyecto en Visual Studio Code.

3.  **Ejecutar:**
    Localiza el archivo principal (por ejemplo, `src/main/java/animacion_3d/AppJoglFull.java`) y ejecútalo desde el IDE.
